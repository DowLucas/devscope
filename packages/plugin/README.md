# DevScope Plugin (local dev copy)

This is a local copy of the DevScope plugin for development convenience.

**For installation, use the standalone repo:**

```bash
claude plugin install github:DowLucas/devscope-plugin
```

See [github.com/DowLucas/devscope-plugin](https://github.com/DowLucas/devscope-plugin) for documentation.

## Local Testing

```bash
claude --plugin-dir packages/plugin
```
